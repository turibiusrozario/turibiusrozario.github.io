Note that this list is rather old. As of June 4th, 2025, I permanently switched
over to Linux (Mint and Kubuntu).

## Cross-Platform Applications

<ul>
  <li>
    <a href="https://apps.ankiweb.net/">Anki</a> (Desktop, Web, Phones):
    Quizlet on steroids. Ask your friends in medical school if you need any convincing.
  </li>
  <li>
    <a href="https://www.blender.org/">Blender</a> (Desktop):
    Free and open-source 3D-Modelling tool with a massive community support.
  </li>
  <li>
    <a href="https://sourceforge.net/projects/briss/">Briss</a> (Desktop):
    Allows cropping contents out of .pdf files. Very useful if you need to
    extract high quality items from these files (e.g. figures from research
        papers).
  </li>
  <li>
    <a href="draw.io">draw.io</a> (Desktop, Web):
    A very easy-to-use and free diagram making software.
  </li>
  <li>
    <a href="https://element.io/">Element</a> (Desktop, Web, Phones):
    Basically Discord, but free, open-source, and ridiculously private. This is
    federated and encrypted, so only you and your recipient have access to
    messages.
  </li>
  <li>
    <a href="https://www.mozilla.org/en-US/firefox/new/"><b>Firefox</b></a>
    (Desktop, Phones): The only major non-Chrome based browser. This is
    especially important since Firefox supports ad-blocking better and there are
    numerous add-ons to significantly upgrade performance (such as returning
    YouTube dislike counts, removing YouTube ads, stronger privacy protection,
    etc.)
  </li>
  <li>
    <a href="https://inkscape.org/">Inkscape</a> (Desktop): Free and open source
    vector drawing application. Extremely useful for drawing diagrams, editing
    PDF files, and vector art.
  </li>
  <li>
    <a href="https://joplinapp.org/help/install">Joplin</a> (Desktop, Phone):
    Minimal yet versatile note taking software with encryption and usability
    with ANY cloud storage (Google, OneDrive, Nextcloud, personal, etc.). This
    is similar to Obsidian, but is free and open sourced.
  </li>
  <li>
    <a href="https://kdenlive.org/">Kdenlive</a> (Desktop): Easy to use and
    minimal video-editing software.
  </li>
</ul>

## Windows-Only Applications

<ul>
<li>
    <a href="https://chocolatey.org/install#individual"><b>Chocolatey</b></a> :
    Application downloader and manager. Similar to `apt install` on Linux.
    Extremely useful for multiple downloads and mass updating applications. In
    fact, many of the applications listed here can be installed via chocolatey.
</li> <li>
    <a href="https://github.com/Code52/carnac">Carnac</a>: Shows keys pressed on
    screen. Useful for doing tutorials.
</li> <li>
    <a href="https://ditto-cp.sourceforge.io/">Ditto</a>: Stores clipboard
    history. Allows easy copy-pasting multiple items.
</li> <li>
    <a href="https://www.voidtools.com/"><b>(Search) Everything</b></a>:
    Literally allows searching everything for something, without ANY noticeable
    loss of computer performance. Highly recommended, especially if you are
    disorganized.
</li> <li>
    <a href="https://handbrake.fr/">Handbrake</a>: Easily change media file
    formats, trim, compress, etc. Very useful if you make videos and need a
    smaller file size out of them.
</li> <li>
    <a href="https://imageglass.org/">Image Glass</a>: A much better and faster
    photo viewer.
</li> <li>
    <a href="https://www.rocksdanister.com/lively/"><b>Lively</b></a>: Live
    desktop wallpapers. This is similar to steam, except that it is more
    minimal, supports audio, and you can make CUSTOM wallpapers. FOSS.
</li> <li>
    <a
        href="https://learn.microsoft.com/en-us/windows/powertoys/"><b>
            PowerToys</b></a>: Significantly upgrades functionality of your
    Windows machine. Adds MANY features ranging from color picker, unit conver
    sion, math, web and computer search, window tiling, and so so so much more,
    all done minmally and with few key presses. This is the one item that should
    be definitely used on a regular basis. FOSS.
</li> <li>
    <a href="https://github.com/QL-Win/QuickLook/releases">Quicklook</a>: Allows
    quickly viewing an item WITHOUT opening it. Useful for quickly viewing a pdf
    document, an image, etc., without opening an application for it.
</li> <li>
    <a href="https://getsharex.com/">ShareX</a>: Screenshot tool with lots of
    features. Can be a bit too many features for some people.
</li> <li>
    <a
        href="https://www.sumatrapdfreader.org/free-pdf-reader">SumatraPDF</a>:
    Very lightweight PDF viewer. FOSS.
</li> <li>
    <a href="https://github.com/lltcggie/waifu2x-caffe">Waifu2x-caffe</a>: Image
    upscaling and denoising. Useful if you download low quality images and wish
    it was better quality. FOSS. There is also a <a
                                                     href="https://www.waifu2x.net/">web
                                                     version</a>.
</li> <li>
    <a href="https://github.com/samhocevar/wincompose/">WinCompose</a>: Allows
    inserting special character very easily (e.g., I can type â‰  by using alt and
    = and / in sequence).
</li> <li>
    <a href="https://github.com/yt-dlp/yt-dlp">yt-dlp</a>: Command line youtube
    (and other) downloader. FOSS.
</li>
</ul>
